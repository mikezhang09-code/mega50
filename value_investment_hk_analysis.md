# 第一部分：价值投资理论基础

## 1.1 本杰明·格雷厄姆：安全边际与内在价值

**"市场短期是一台投票机，但长期是一台称重机。"** ——格雷厄姆

### 格雷厄姆的核心投资原则

| 原则 | 具体标准 | 筛选指标 |
|------|----------|----------|
| **内在价值** | 股票价值应基于公司盈利能力，股价围绕内在价值波动 | DCF估值、NPV |
| **安全边际** | 买入价应显著低于内在价值（通常折扣≥30%） | 股价/内在价值 < 0.7 |
| **盈利稳定性** | 过去10年盈利基本稳定，无连续2年亏损 | 10年EPS趋势 |
| **股息记录** | 连续20年支付股息 | 股息稳定性 |
| **市盈率（PE）** | PE < 15倍为理想，< 20倍可接受 | PE < 20 |
| **市净率（PB）** | PB < 1.5倍，< 1倍为极佳 | PB < 1.5 |
| **资产负债率** | < 50%（金融类公司除外） | 负债/资产 < 50% |

### 格雷厄姆选股公式
```
格雷厄姆内在价值 = √(15 × 每股收益 × 8.5 × 每股净资产)
安全边际买入价 = 内在价值 × 0.7
```

## 1.2 沃伦·巴菲特：护城河与优质公司

**"如果你不愿意拥有一只股票十年，那就不要考虑持有它十分钟。"** ——巴菲特

### 巴菲特的核心投资原则

| 原则 | 具体标准 | 筛选指标 |
|------|----------|----------|
| **优质公司（Wonderful Companies）** | 具有持久竞争优势、高ROE、低负债 | ROE > 15%，毛利率稳定 |
| **经济护城河（Economic Moat）** | 竞争对手无法复制的独特优势 | 品牌、专利、网络效应、成本优势 |
| **管理层质量** | 诚实、有能力、以股东利益为导向 | 资本配置记录、回购/分红政策 |
| **自由现金流** | 产生超过业务再投资需求的现金 | FCF > 净利润的80% |
| **合理价格** | 以合理或低估价格买入优质公司（"合理价格买优质公司"） | PE低于行业平均 |
| **盈利能力** | 持续的高ROE，代表资本高效运作 | 5年平均ROE > 15% |
| **稳定性** | 业务简单易懂，盈利可预测 | 盈利波动性低 |

### 巴菲特护城河的五大来源
1. **无形资产**：品牌、专利、特许经营权（如香港交易所的垄断性牌照）
2. **成本优势**：规模经济、网络效应（如腾讯的社交网络）
3. **转换成本**：客户粘性高的产品（如金融保险产品）
4. **网络效应**：用户越多价值越大的平台（如微信）
5. **高效规模**：区域垄断带来的有限竞争（如电力、基建）

## 1.3 格雷厄姆与巴菲特的异同及综合应用

| 维度 | 格雷厄姆 | 巴菲特 | 本报告综合应用 |
|------|----------|--------|----------------|
| 核心关注 | 安全边际（价格） | 商业质量（公司） | 安全边际 + 优质公司 |
| PE阈值 | < 15倍 | 合理即可（强调护城河） | < 20倍 |
| PB阈值 | < 1.5倍 | < 3倍（优质公司） | < 3倍 |
| ROE要求 | 无硬性要求 | > 15% | > 15% |
| 股息要求 | 连续20年分红 | 重视但非必须 | 连续5年稳定分红 |
| 持仓周期 | 中期（价值回归后卖出） | 长期（10年以上） | 3-5年中期持有 |
| 分散程度 | 高度分散（30+只） | 集中（10只以内） | 10只核心持仓 |

# 第二部分：港股选股标准体系

## 2.1 量化筛选指标

结合格雷厄姆与巴菲特的原则，本报告建立以下量化筛选标准：

### 基础量化指标

| 指标 | 筛选标准 | 价值投资原则 |
|------|----------|-------------|
| **ROE（净资产收益率）** | 5年平均 > 12%，近1年 > 10% | 巴菲特：资本效率 |
| **PE（市盈率）** | < 25倍，理想 < 15倍 | 格雷厄姆：估值安全 |
| **PB（市净率）** | < 2.5倍，理想 < 1.5倍 | 格雷厄姆：资产安全 |
| **股息率** | > 2.5%，理想 > 3.5% | 格雷厄姆+巴菲特：现金回报 |
| **净利润增长率** | 5年CAGR > 5% | 巴菲特：成长性 |
| **资产负债率** | < 60%（金融类 < 80%） | 格雷厄姆：财务健康 |
| **流动比率** | > 1.2倍 | 格雷厄姆：流动性安全 |

### 定性评估指标

| 指标 | 评估内容 |
|------|----------|
| **经济护城河** | 5项护城河来源（品牌/专利/网络效应/转换成本/规模经济） |
| **行业地位** | 市场占有率、定价能力、行业门槛 |
| **管理层质量** | 资本配置能力、分红回购记录 |
| **商业模式** | 盈利可预测性、资本支出需求 |
| **治理结构** | 股东权益保护、信息披露质量 |
