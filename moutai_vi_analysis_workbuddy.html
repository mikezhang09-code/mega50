01 Executive Summary / 执行摘要

03 Moat Analysis / 护城河分析
Business Classification
● Franchise Business / 特许经营型企业
● STABLE / 稳定 — 护城河未见收窄迹象

护城河评分 13.5 / 15 — Wide Moat

无形资产 / Intangible Assets3 / 3

茅台是中国白酒文化皇冠上的明珠。品牌创立逾70年，拥有国家地理标志保护、千年酿酒工艺非物质文化遗产。消费者愿意为"茅台"二字支付10倍于普通白酒的价格。这是全球最强的品牌护城河之一，与Coca-Cola、Louis Vuitton同级。

成本优势 / Cost Advantage2.5 / 3

赤水河谷独特的风土条件（水、土壤、微生物）不可复制。茅台镇核心产区产量有限（约5.6万吨/年），天然稀缺性创造了结构性成本优势。吨成本约¥2.6万，终端售价¥2,000+/瓶，毛利率91%+。

转换成本 / Switching Costs2.5 / 3

商务宴请、礼品馈赠场景中，茅台是"最安全的选择"。品牌社会属性强大——送茅台=面子，换其他品牌有社交风险。这种心理转换成本极高，是护城河的重要组成部分。

网络效应 / Network Effects2 / 3

虽然白酒不像互联网产品有传统网络效应，但茅台在社交场景中形成了"共识网络"——越多人认可茅台的价值，其社交货币属性越强。收藏市场也形成正反馈：价格越高，收藏意愿越强。

有效规模 / Efficient Scale3 / 3

茅台镇核心产区仅约15平方公里，茅台在此拥有绝对主导地位。新进入者即使有资金也无法获得同等风土条件。2025年茅台酒产量约56,000吨，远低于需求，天然形成了有效规模壁垒。

护城河雷达图

Munger 逆向思维 / Inversion Stress Test

"什么能在10年内摧毁茅台的护城河？"

① 年轻人完全拒绝白酒 → 低概率，高端社交场景不会消失

② 茅台自身品质事故 → 低概率，严格的质量管控体系

③ 重大政策禁酒 → 极低概率，白酒是地方经济支柱

④ 造假泛滥摧毁信任 → 可控，官方防伪+数字化溯源

结论：威胁均为低概率且可控，护城河预计维持20年以上。
